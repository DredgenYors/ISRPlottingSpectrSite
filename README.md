Main Repo Storing Files for the NJIT ISR Plotting Website
Contains files related to plotting, website navigation and design, as well as comparisons to other ISR models

1. EEVERSION:
   Contains all files related to the EE ISR approach
2. EEvsISR:
   Various testing files to compare the NJIT to the EE ISR
3. __pycache__:
   Cache storing intermediate, frequently used ISR comparisons
4. Static:
   js and styles for the main ISR site
5. templates:
   html files for the ISR site
6. ISR_spectra (1).ipynb:
   Jupyter Notebook with individual functions and plotting logic that was integrated into the ISR site
7. spectraapp.py:
   Main python file that controls the website
8. spectratesting_with_marcos_plot.py:
   Main python file for EE vs NJIT ISR testing
9. spectratestingmodified_chirag_edit.py:
    Main python file containing NJIT ISR spectra calculation logic
